Como Jogar - Modo Clássico

Objetivo

Descubra qual é o time de futebol brasileiro secreto! Teste seus conhecimentos, faça palpites e use as dicas fornecidas para chegar à resposta correta.

Como Jogar

  • Digite o nome de um time brasileiro.
  • Receba dicas baseadas nas características do time.
  • Use o código de cores para entender se está perto ou longe da resposta.

Código de Cores

Verde = Correto
Amarelo = Parcialmente correto
Vermelho = Incorreto

Dicas Especiais

  • 4 tentativas erradas: Você recebe uma dica sobre a série do time (A, B ou C).
  • 7 tentativas erradas: Você recebe uma dica sobre o mascote do time.

Todas as regras podem ser atualizadas para aprimorar a experiência de jogo. Veja a seção de atualizações.
